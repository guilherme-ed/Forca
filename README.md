# 🎮 Jogo da Forca com Streamlit

Um jogo da forca interativo feito com Python e Streamlit, com interface no navegador, categorias de palavras e placar de vitórias.

## ✨ Funcionalidades

- 4 categorias: Animais, Países, Tecnologia e Esportes
- Dicas para cada palavra
- Desenho da forca atualizado a cada erro
- Placar de vitórias e derrotas
- Interface visual com tema retrô

## 🚀 Como rodar localmente

### 1. Clone o repositório
```bash
git clone https://github.com/https://github.com/guilherme-ed/forca-game.git
cd forca-game
```

### 2. Instale as dependências
```bash
pip install -r requirements.txt
```

### 3. Rode o app
```bash
streamlit run app.py
```

O app abrirá automaticamente em `http://localhost:8501`

## 🛠️ Tecnologias

- [Python 3.10+](https://www.python.org/)
- [Streamlit](https://streamlit.io/)
